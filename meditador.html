Open Source y Accesible

El Meditador de Bolsillo no es solo una idea: es un dispositivo funcional, probado y con un alto nivel de desarrollo técnico. Estamos trabajando para que muy pronto puedas comprar el tuyo.

También creemos que este proyecto le pertenece a todos. Por eso liberamos el diseño y el código como open source. No solo para que puedas ver cómo está hecho, sino para que puedas adaptarlo, mejorarlo o incluso construir tu propia versión.
